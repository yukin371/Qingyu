## 📖 使用指南

### 快速开始

1. **启动服务**
   ```bash
   # 后端
   cd Qingyu_backend && go run cmd/server/main.go
   
   # 前端
   cd Qingyu && npm run dev
   ```

2. **访问测试工具**
   - URL: http://localhost:5173/api-test
   - 或点击导航栏"API测试"

3. **开始测试**
   - 详见：[API测试快速开始.md](./API测试快速开始.md)
